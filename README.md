Sleep Pattern Recognition in Pediatric ADHD

This project explores the relationship between sleep schedules and Attention-Deficit/Hyperactivity Disorder (ADHD) in children using computational and machine learning approaches. The aim is to identify ADHD-associated sleep irregularities by analyzing patterns in sleep timing, duration, and variability.
Sleep-related features such as sleep onset latency, total sleep time, wake after sleep onset, night-to-night variability, and circadian rhythm disruptions are extracted from behavioral and physiological sleep data. These features are processed as time-series data and used to train supervised machine learning models to distinguish ADHD-specific sleep signatures from typical sleep patterns.
